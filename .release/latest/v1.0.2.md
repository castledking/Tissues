## What's Changed

Tissues (Track Issues) is a browser extension that helps you track the status of issues across repositories you've forked on GitHub. It adds a simple □ → ⏳ → ✅ status indicator to each issue row in GitHub's issue lists, so you can keep track of what you need to work on at a glance.

### Features
- **One-click issue tracking** — Click the indicator on any issue to cycle through To-Do (□), In Progress (⏳), and Done (✅) states
- **Fork-only activation** — The extension only activates on repositories you've forked, keeping unrelated repos clean
- **No API token required** — Fork detection works by checking GitHub's /fork page, no personal access token needed
- **Persistent state** — Issue states are saved locally and survive page reloads and navigation
- **React-aware** — Uses per-row MutationObservers to maintain indicators across GitHub's React re-renders
- **Cross-browser** — Available for Chrome/Chromium and Firefox

### Bug Fixes
- Fixed tissues indicator alignment on closed issues — now correctly appears on the rightmost side (after PR count, comment count, and assignees), matching the layout on open issues
- Fixed Firefox extension ID (now `tissues@castled.codes`)

### Technical
- Cached fork check result to avoid redundant network requests
- Fallback URL-based repo detection when meta tags are unavailable
