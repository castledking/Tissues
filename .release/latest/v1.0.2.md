## What's Changed

### Bug Fixes
- Fixed tissues indicator alignment on closed issues — now correctly appears on the rightmost side (after PR count, comment count, and assignees), matching the layout on open issues
- Added per-row MutationObserver to maintain indicators across React re-renders

### New Features
- Extension now only activates on repositories you've forked — no more clutter on unrelated repos
- Ownership detection skips repos you own outright
- Fork detection via GitHub's `/fork` page (no API token required)

### Technical
- Cached fork check result to avoid redundant network requests
- Fallback URL-based repo detection when meta tags are unavailable
